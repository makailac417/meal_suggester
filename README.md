# meal_suggester

A simple command-line tool that suggests a random meal based on how much time you have to cook and your budget. It displays the ingredients for each suggestion and lets you keep re-rolling until something sounds good.

Features

Filter meals by cooking time and budget tier (cheap, medium, expensive)
View a full ingredients list for every suggestion
Re-roll as many times as you want — already-seen meals won't repeat
Input validation with helpful re-prompts on bad input
